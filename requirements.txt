# AI Resume Analyzer - Installation Requirements

## Prerequisites
- Node.js >= 20.0.0
- npm >= 8.0.0

## Installation Command
npm install

## Setup Instructions
1. Clone the repository
2. Run: npm install
3. Run: npm run dev (for development)
4. Run: npm run build (for production build)
5. Run: npm start (to serve production build)

## Notes
- All dependencies are managed via package.json
- Use npm install to install all required packages automatically
- No manual installation of individual packages needed
